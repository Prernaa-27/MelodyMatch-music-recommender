# MelodyMatch

MelodyMatch is a Python-based music recommendation system that suggests songs based on language, mood, and era using MySQL database integration.

## Features
• Recommend songs based on user mood  
• Filter by language (Hindi, English, Punjabi)  
• Filter by era (90s, 2000s, 2010s, Trending)  
• Create your own music directory  
• Add, update, delete songs  

## Technologies Used
• Python  
• MySQL  
• MySQL Connector  
• VS Code  

## How to Run

1. Install requirements
pip install mysql-connector-python

2. Setup MySQL database
Run database_schema.sql in MySQL Workbench

3. Run the program
python CS_PROJECT.py
